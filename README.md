## Installation

Get code using

```
git clone https://github.com/syahrulafrizal/jmc-challange-code.git
```

__Step 2__

Install the necessary libraries (this proyect use Flutter 3.24.3)

```
flutter pub get
```

__Step 3__

Run the app

```bash
flutter run
```
